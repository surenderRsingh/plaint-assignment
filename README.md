# Pre requisite
 - Node version > 10
 - Chrome version 86

 ## Initial set up
  > Download or clone project
  > Download all required modules
      ```
      npm install
      ```
  > execute test cases
    ```
    npx wdio wdio.config.js
    ```
  > Reports are saved at ** Reports/html ** folder

  > For changing the chrome version - change _ chromedriver _ value in package.json
